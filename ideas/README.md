# Ideas

**Status:** Open collection of exploratory notes — not repository claims.

This directory holds small, self-contained exploratory notes. Each note captures one observation, question, hypothesis, analogy, or disturbance before it needs a stable place in a theory, chapter, experiment, or concept registry.

Notes are intentionally small — usually around 100–200 words. They preserve enough context to recover why an idea mattered, name what remains open, and link to genuine neighbours when those exist. They are not miniature essays and do not need a category, mechanism, falsifier, or destination merely to be recorded.

## Working rhythm

`note → connections → cluster → essay / experiment / chapter → synthesis`

The arrows are possibilities, not obligations. Most work happens by accumulation: when several notes keep pointing to one another, a cluster may have appeared. Only then is it worth asking whether a larger artifact is needed.

## Minimal shape

Each note should contain:

- one observation, question, or hypothesis;
- a little context;
- an open question or tension;
- optional links to related notes or repository material.

Use `YYYY-MM-DD-short-title.md`. The date preserves the encounter; the title keeps the note readable. No numbering, taxonomy, or promotion queue is required.
