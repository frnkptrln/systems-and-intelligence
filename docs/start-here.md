# Start Here

This repository is a living research system about intelligence as constrained dynamics.

Its central claim is not that "everything is connected." The sharper claim is: **an intelligent system remains viable only when local optimization is constrained by the carrying capacity of its substrate and regulators.** When action entropy outruns dissipation, feedback, or human review capacity, the system becomes brittle even if its local predictions improve.

The repository has two modes.

**Synthesis** contains claims about reality: formal arguments, definitions, models, and publication-facing summaries. Start here if you want the strongest version of the theory:

- [Synthesis Overview](synthesis.md)
- [Core Claims](theory/core-claims.md)
- [Emergence Manifesto v1.2](theory/emergence-manifesto-v1.2.md)
- [Thermodynamics of Orchestration](theory/thermodynamics-of-orchestration.md)
- [Substrate Veto](theory/substrate-veto-thermodynamics.md)
- [Simulation to Theory Map](theory/simulation-theory-map.md)

**Thinking Space** contains exploratory material: architecture logs, simulations, stress tests, fiction, and unfinished design moves. Start here if you want to see the system being tested:

- [Thinking Space Overview](thinking-space.md)
- [Simulation to Theory Map](theory/simulation-theory-map.md)
- [Architecture Logs](logs/README.md)

One useful reading path:

1. Read the [Emergence Manifesto](theory/emergence-manifesto-v1.2.md) for the claim set.
2. Read the [TEO Framework](theory/teo-framework/README.md) for the constraint model.
3. Run or inspect one proof artifact, such as the [AI Alignment Veto](simulation-models/ai-alignment-veto/README.md) or [Symbiotic Breathing](simulation-models/symbiotic-breathing/README.md).
4. Use the [Architecture Logs](logs/README.md) to see how the formal constraints translate into governance and deployment design.

Do not read this repository as a finished textbook. Read it as a structured research workspace: claims are separated from experiments, and speculative extensions are expected to remain challengeable.
